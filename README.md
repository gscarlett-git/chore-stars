# ⭐ Chore Stars

A fun chore tracking app for kids, with a parent management portal. Built with React, Node.js, MySQL, and Docker — designed to run behind Traefik on a home server.

## Features
- 👧 Per-child chore tracking with star rewards
- 📅 Flexible scheduling: daily, school days, weekends, specific days, monthly
- 🎁 Reward nomination and redemption system
- 👨‍👩‍👧 Parent portal with full management controls
- 🎉 Confetti celebrations on chore completion
- 📱 Mobile and desktop friendly

## Stack
- **Frontend**: React 18, served via Nginx
- **Backend**: Node.js / Express
- **Database**: MySQL 8
- **Reverse proxy**: Traefik v3 with Cloudflare SSL
